This is the demo of hugo-theme-monochrome.

Features

  1. 1. Clean UI with small resources. Monochrome has clean UI design and can remove unused resources in production environment.
  2. 2. Responsive layout. Adapt to different viewpoint, from desktop to mobile.
  3. 3. Dark mode. Low-light UI.
  4. 4. Multiple layouts built-in. See Demo.
  5. 5. Nested navbar and Multilingual mode. Make good use of hugo's well-designed structure.
  6. 6. Syntax Highlighting (with copy feature). Integrate both server-side solution (Chroma, hugo built-in) and client-side solution (Prism.js, for someone who need more languages supported).
  7. 7. Site Search. Powered by FlexSearch.
  8. 8. SEO-friendly. Easily change your author, description, keywords, open graph, twitter card metadata.
  9. 9. Support mathematical notation. Powered by MathJax.
  10. 10. Useful shortcodes. See Document.
