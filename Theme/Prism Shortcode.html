Prism

Cara menggunakan Prism Shortcode untuk sintaks penyorotan blok kode.

Prism Shortcode

Ketika Prism diaktifkan di config.toml, sintaks penyorotan juga dapat dicapai menggunakan shortcode Prism berpasangan. Bahasa kode dideklarasikan menggunakan parameter lang:

<{{ prism lang="html" }}>
<html>
  <head>
    <title>@My cool new product</title>
  </head>
  <body>
    <!-- Use action="/create-checkout-session.php" if your server is PHP based. -->
    <form action="/create-checkout-session" method="POST">
      <button type="submit">Checkout</button>
    </form>
  </body>
</html>
<{{ /prism }}>

Semua blok kode yang disorot oleh Prism menampilkan plugin Tombol Salin ke Clipboard. Arahkan kursor ke (atau ketuk pada perangkat seluler) contoh di atas dan tombol salin muncul di sudut kanan atas blok kode. Klik tombol ini untuk menyalin kode ke clipboard Anda.

Line Highlighting

Shortcode Prism dapat menyorot baris tertentu dan/atau rentang baris dalam blok kode menggunakan parameter line:

<{{ prism lang="html" line="2-4,6" }}>
<html>
  <head>
    <title>...</title>
  </head>
<{{ /prism }}>
