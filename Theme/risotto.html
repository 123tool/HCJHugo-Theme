# About

Ditulis dalam Go, Hugo adalah generator situs statis sumber terbuka yang tersedia di bawah Lisensi Apache 2.0. Hugo mendukung tipe file data TOML, YAML, dan JSON, Markdown dan file konten HTML dan menggunakan shortcode untuk menambahkan konten yang kaya. Fitur menonjol lainnya adalah taksonomi, mode multibahasa, pemrosesan gambar, format output kustom, minifikasi HTML/CSS/JS dan dukungan untuk alur kerja Sass SCSS.

Hugo memanfaatkan berbagai proyek sumber terbuka termasuk:

  • https://github.com/vwin/goldmark
  • https://github.com/alecthomas/chroma
  • https://github.com/muesli/smartcrop
  • https://github.com/spf13/cobra
  • https://github.com/spf13/viper

Hugo ideal untuk blog, situs web perusahaan, portofolio kreatif, majalah online, aplikasi satu halaman atau bahkan situs web dengan ribuan halaman.

Hugo adalah untuk orang-orang yang ingin mengkodekan situs web mereka sendiri secara manual tanpa khawatir tentang pengaturan runtime yang rumit, dependensi, dan basis data.

Pelajari lebih lanjut dan berkontribusi di GitHub.
