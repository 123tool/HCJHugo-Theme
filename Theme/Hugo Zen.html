Hugo

Make the Web Fun Again

Memperkenalkan Hugo, sebuah ide baru untuk membuat pembuatan situs web menjadi sederhana kembali. Hugo bekerja secara fleksibel dengan banyak format, dan sangat ideal untuk blog, dokumen, portofolio, dan banyak lagi. Kecepatan Hugo mendorong kreativitas—ini membuat pembuatan situs web menyenangkan kembali.

Run Anywhere

Hugo kemungkinan besar adalah perangkat lunak termudah untuk diinstal yang pernah Anda gunakan: cukup unduh dan jalankan! Hugo tidak bergantung pada hak administratif, basis data, waktu proses, penerjemah, atau perpustakaan eksternal. Situs yang dibangun dengan Hugo dapat diterapkan di S3, GitHub Pages, Dropbox, atau host web apa pun.
