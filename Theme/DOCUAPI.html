Introduction

Selamat datang di Kittn API! Anda dapat menggunakan API kami untuk mengakses endpoint Kittn API, yang dapat memberikan informasi tentang berbagai kucing, anak kucing, dan ras di database kami.

Kami memiliki language bindings dalam Shell, Ruby, dan Python! Anda dapat melihat contoh kode di area gelap di sebelah kanan, dan Anda dapat mengganti bahasa pemrograman dari contoh tersebut dengan tab di pojok kanan atas.

Halaman dokumentasi API contoh ini dibuat dengan DocuAPI, tema dokumentasi multibahasa untuk static site generator Hugo.

Authentication

Kittn menggunakan kunci API untuk memberikan akses ke API. Anda dapat mendaftarkan kunci Kittn API baru di developer portal kami.

Kittn mengharapkan kunci API disertakan dalam semua permintaan API ke server dalam header yang terlihat seperti berikut:

Authorization: meowmeowmeow
Anda harus mengganti meowmeowmeow dengan kunci API pribadi Anda.

To authorize, use this code:
package main

import "github.com/bep/kittn/auth"

func main() {
    api := auth.Authorize("meowmeowmeow")

    // Just to make it compile
    _ = api
}
# With shell, you can just pass the correct header with each request
curl "api_endpoint_here" \
  -H "Authorization: meowmeowmeow"
Pastikan untuk mengganti meowmeowmeow dengan kunci API Anda.
