## HTML Web Theme Clones Collection

​**Kumpulan klon tema web populer (terutama dari ekosistem Hugo, Jekyll, dan kustom) yang dibangun menggunakan HTML5, Tailwind CSS, dan Vanilla JavaScript dalam format Single-File (Satu File per Tema).**

**Proyek ini dibuat untuk menunjukkan kemudahan pembuatan antarmuka (UI) modern yang cepat, responsif, dan tanpa ketergantungan framework berat.**
